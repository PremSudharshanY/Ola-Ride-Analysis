# 🚖 Ola Ride Analysis

## 📌 Overview
This project analyzes ride booking data using SQL, Python, and Power BI.

## 📊 Key Numbers
- Total Bookings: 600K
- Successful Rides: 382K
- Cancellation Rate: 22%
- Ratings: Around 4.0

## 🛠 Tools Used
- SQL
- Python
- Power BI

## 📈 Insights
- Prime SUV has highest rating
- UPI & Cash most used
- High cancellations due to plan changes
